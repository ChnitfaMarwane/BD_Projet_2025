----Projet Base de donnée---

Ce projet consiste à créer une base de données de réservation d’hôtels, 
puis à la manipuler via des requêtes SQL et en algèbre relationnelle.
Dans une seconde phase, une interface web est développée avec Python, SQLite, et Streamlit 
pour permettre la consultation, l’ajout de clients et de réservations, 
ainsi que la recherche de chambres disponibles.
